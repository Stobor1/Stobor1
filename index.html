Bienvenue sur le projet PI_RATE

Il s'agit d'un jeu où l'on doit ouvrir le coffre aux trésors. Il y a deux coffres. L'un contient les épreuves et l'autre le trésor. Il a plusieurs épreuves au jeu. Lorsque les épreuves sont réussies, le petit coffre s'ouvre. Chacune des épreuves utilisent différentes composantes. Nous expliquerons chacune des composantes utilisées ainsi que l'application mobile et les instructions de chaque jeu. Pour l'ensemble du projet, nous avons utilisé deux raspberry pi et un arduino. Ce projet a été réalisé dans le cadre de trois cours d'informatique 420-614-RI OBJETS CONNECTÉS, 420-616-RI PROJET D'INTÉGRATION - PHASE II et 420-636-RI PROGRAMMATION D'APPLICATIONS MOBILES.

Raspberry Pi

Deux Raspberry Pi ont été utilisés dans le cadre de notre projet. L'un est dans le petit coffre et l'autre dans le gros coffre. Le Raspberry Pi sert a connecter des objets comme des led, des bouttons, des moteurs et même l'Arduino.

Arduino

Nous avons utilisé un Arduino connecté à un Raspberry Pi afin d'avoir plus de prise GPIO. L'Arduino se retrouve dans le gros coffre connecté au PI par câble USB.

Petit coffre

Le petit coffre comprend un Raspberry Pi, une batterie 5V afin de l'alimenter, une led, des buzzers, un lecteur RFID et un servo moteur qui sert à vérouiller le coffre.

Gros coffre

Le gros coffre comprend un Arduino connecté à un Raspberry Pi, un power supply d'ordinateur ainsi que tous les challenges. Un lecteur de carte RFID permet de scanner une carte afin de déverrouiller le petit coffre. Il y a aussi un clavier (keypad) afin de faire une fausse synchronisation.

Fonctionnement

Le fonctionnement du système. On utilise un PI dans le gros coffre comme broker. Le broker reçoit et envoi les requêtes MQTT à ceux qui souscrivent aux topics publiés. Tous les challenges sont programmés en python avec des threads pour certaines fonctionnalités. L'application mobile a été développé en Java.

Drapeau

Ce jeu consiste à faire le mélange de couleur demandé à l'aide de boutons. Le jeu comporte 3 led de couleurs différentes et trois boutons associés aux couleurs correspondantes.

Assault

Ce jeu consiste à utiliser un potentiomètre et d'atteindre une valeur aléatoire. Le potentiomètre est un élément analogique qui requiert l'utilisation d'une puce ADS7830 qui convertie le signal analogique vers un signal numérique.

Pompier

Ce jeu consiste à déplacer un personnage vers l'extincteur à l'aide d'un joystick. Le joystick est un élément analogique qui requiert aussi l'utilisation de la puce ADS7830.

CAA

Ce jeu consiste à connecter un câble d'un symbole à un autre. Nous avons utilisé des câbles de type RCA.

Bi-Nerf

Le jeu consiste à faire un nombre binaire à l'aide d'interrupteurs. Nous avons utilisé un Arduino comme extension du Raspberry PI et la bibliothèque pyfirmata.

Application mobile

L'application mobile a été développé avec Android Studio en langage Java.
